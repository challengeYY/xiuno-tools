### Discuz!X 3.x 转换 XiunoBBS 4.x
------

**(本程序不支持迁移文件)**   

**不作处理的表**
- cache - 缓存表，用来保存临时数据
- forum_access 版块访问规则
- kv - 持久的 key value 数据存储
- modlog -  版主操作日志
- queue - 临时队列，用来保存临时数据
- session - session 表
- session_data 
- table_day - 系统表

**Xiuno 表解析参考:** https://www.kancloud.cn/xiuno/bbs4/214347   
**Discuz 表解析参考:** http://discuzt.cr180.com/discuzcode-db.html
